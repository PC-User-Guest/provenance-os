Final Project Report — Provenance OS (Draft)
=============================================

Summary
-------
Provenance OS demonstrates a compact production-ready static UI for a Computational Evidence Workspace. The redesign emphasizes evidence-first interactions, a permanent temporal rail, and clear trust semantics.

Achievements
------------
- Implemented three permanent navigation primitives (Space, Time, Trust).
- Built a static prod bundle and deployed to Vercel.
- CI Lighthouse job added to generate performance and accessibility artifacts.
