PR Tasks — Checklist
=====================

Use this checklist when opening a Pull Request against `main`.

- [ ] Title follows: Category: short description (e.g., "Docs: add project docs")
- [ ] Include screenshots for UI changes
- [ ] Run local static server and verify no console errors
- [ ] Confirm keyboard navigation works for topology and timeline
- [ ] Update `DocumentedChanges.md` with a short summary
- [ ] Add Lighthouse notes if relevant
