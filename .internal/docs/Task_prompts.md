Task Prompts — Developer Notes
==============================

Short prompts to help reproduce common edits or tasks.

- "Rebuild UI per Web UX v2": Update `index.html` and `styles.css` to follow the three-column layout and temporal rail. Preserve DOM IDs.
- "Accessibility pass": Run Lighthouse, fix contrast for small labels, ensure keyboard focus on SVG nodes.
- "Generate release": Run CI, download `lighthouse.json`, attach to release notes.
